Flexi NG (GGSN)

The FlexiPlatform software platform is based on Windriver Linux. ATCA shelf used by Flexi NG contains 16 blade slots. Used in Octeon MIPS and Intel x86 HW environments. For each blade in the ATCA shelf, a single hub blade provides a 1 Gbps base interface and a 10 Gbps fabric interface. Environment is intended for running fast path applications that require most out of the packet forwarding and acceleration hardware.

SGSN NS ATCA

Platform software platform is based on proprietary DMX operating system and WR Linux operating system. HW contains up to 3 ATCA shelfs. ATCA shelf used by Flexi NS contains 16 blade slots. Used in Octeon MIPS and Intel x86 HW environments. For each blade in the ATCA shelf, a single hub blade provides a 1 Gbps base interface and a 10 Gbps fabric interface. Virtualization makes it possible to implement several logical functional units as virtual machines into the same physical functional unit.

Flexi ISN (GGSN)

FlexiServer hardware platform consists up to two shelfs. Each shelf used by Flexi ISN contains 14 blade slots. The Flexi ISN uses the Nokia Ipsilon Router Operating system (IPSO) software platform based on BSD. The Flexi ISN supports both Radius and Diameter control interfaces. Flexi ISN acts as a policy enforcement point, enabling to control the QoS and session bitrates, service access and service usage, on per subscriber basis.

Cisco ASR 900 Series

This full-featured Layer 2 and Layer 3 switch router supports broadband aggregation for delivering "any-play" services. Deployed as a pre-aggregation platform for mobile backhaul, the Cisco ASR 900 Series router can aggregate cell sites and use Multiprotocol Label Switching (MPLS) as a transport for Radio Access Network (RAN) backhaul traffic.

SGSN DX-200

The Flexi NS - SGSN is based on modular software and hardware architecture. The architectural solution of the Flexi NS - SGSN provides high capacity with dedicated processing power for various 2G and 3G GPRS tasks combined with carrier-class availability. The Flexi NS - SGSN supports balanced packet processing and subscriber/PDP context capacity with flexible connectivity options and capacity.

Checkpoint firewalls

The Security Platform combines the power of the Nokia IPSO operating system with the Secure Access System and firewall applications. Security Platform is a high-end, multi port security platform that is ideally suited for the enterprise data center. Series Security Platforms support an encryption accelerator card to further enhance VPN performance.
